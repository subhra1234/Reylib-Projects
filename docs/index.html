The source code of all the projects are available here

To build the source code on your system, you will need the following installed,

  • C++ compiler (such as gcc or g++)
  • Raylib Library

For mobile devices. Follow the keymap below

  • Tap once to create items (like pendulum)
  • Tap with two fingers to start simulation (if not started)
  • Tap with three fingers to delete items
note: touch will be registered if touched on black area only.
